# 企业员工管理系统 python134

## 项目概述

企业员工管理系统是基于 Django 框架开发的 Python 应用，主要面向企业内部管理，用于提高工作效率和员工信息管理的便捷性。

## 技术栈

- 后端：Python（Django 框架）
- 前端：Web 界面
- 数据库：MySQL

## 功能模块

系统包括但不限于以下功能模块：

- 员工信息管理
- 部门管理
- 职位管理
- 工资管理
- 请假管理

## 系统角色

- 系统管理员
- 部门经理
